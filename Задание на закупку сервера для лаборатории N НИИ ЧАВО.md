# Задание на закупку сервера для лаборатории N НИИ ЧАВО

## Введение

В связи с увеличением объемов геномных данных и переходом сотрудников на удаленную работу в условиях болезни Y, необходимо модернизировать серверное оборудование лаборатории.

## Требования к серверу

### 1. Оперативная память

- Сотрудники, занимающиеся тяжелыми вычислениями (3 человека):
  - Необходимо 100 ГБ ОЗУ x 3 сотрудников = **300 ГБ ОЗУ**.
- Сотрудники офисного отдела (7 человек):
  - Необходимо 32 ГБ ОЗУ x 7 сотрудников = **224 ГБ ОЗУ**.
- **10%** сверху на гипервизор

**Итого:**

- **(300 ГБ + 224 ГБ) х 1.1 = 577 ГБ ОЗУ**. => чуть больше 9 х 64 ГБ ОЗУ

### 2. Хранилище данных

- Лаборатория производит примерно 50 ГБ данных в месяц.
- **50 ГБ x 12 месяцев x 3 года = 1800 ГБ** (или 1.8 ТБ).
- Нужно предусмотреть место под гипервизор, ОС и ППО.

### 3. Бюджет

- Выделенный бюджет на сервер: **700,000 рублей**.
